# Challenge CSS

## UI Kit

Este challenge consiste en maquetar en HTML y CSS (nada de JS) la web de [UI Kit](https://www.figma.com/file/gsN3ijVw3LCefjCpjuVpli/Landing-Page-UI-Kit---Challenge?type=design&node-id=0-1&mode=design&t=GmAU1dFpxkFklBMX-0), a partir del figma proporcionado: versión mobile y versión desktop.

Se utilizará SASS para los estilos.

Requisitos:

- HTML semántico y validado con el validador HTML.
- CSS con BEM y validado con el validador CSS.
- Enfoque mobile-first.
- Media queries (un solo breakpoint).
- Hay que desplegar la web
  - en gitHub pages
  - alternativamente en Vercel / Netlify.

Opción: Hazlo en parejas (o grupos)

- Hay que proteger en GitHub la rama principal para que no se pueda volcar nada en ella sin hacer una Pull Request.
- Las Pull Requests no se pueden mergear sin que haya como mínimo un approval.
